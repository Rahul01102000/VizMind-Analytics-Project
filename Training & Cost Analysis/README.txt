# Training & Cost Analysis | Power BI Project

## 📌 Project Overview
This project analyzes employee training and associated costs using interactive dashboards built in Power BI. The goal is to provide insights into training participation and budget utilization for better decision-making.

## 🛠️ Tools Used
- Power BI
- Microsoft Excel

## 📂 Contents
- `Training_&_Cost_Analysis.pbix`: Main Power BI dashboard file
- `Data/`: Source Excel files
- `Report/`: Full project report with findings and recommendations
- `Screenshots/`: Dashboard snapshots for quick reference

## 📈 Dashboards Included
- Training Participation Overview
- Cost Analysis
