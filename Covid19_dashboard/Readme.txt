🦠 COVID-19 Dashboard
📌 Project Overview

The COVID-19 Dashboard is an interactive visualization project built using Tableau. It provides a clear and insightful view of the COVID-19 pandemic’s impact across regions, enabling users to analyze confirmed cases, recoveries, deaths, and trends over time.

This project demonstrates skills in data visualization, storytelling, and dashboard design using Tableau.

⚙️ Features

Interactive Dashboard with filters and drill-down options.

KPIs (Key Performance Indicators):

Total Confirmed Cases

Total Deaths

Total Recovered

Trend Analysis: Time-series charts showing how cases evolved.

Geographic View: Map visualization highlighting country/region-wise COVID spread.

Comparative Analysis: Ability to compare multiple regions over time.

🗂️ Data Sources

COVID-19 dataset containing country-wise daily data on confirmed, recovered, and death cases.

Data pre-processed and loaded into Tableau for visualization.

🛠️ Tools & Technologies

Tableau → Dashboard design and visualization

Excel / CSV (input data) → Dataset storage

📊 Dashboard Insights

Growth trends of COVID-19 cases globally.

Countries with the highest number of confirmed cases.

Mortality and recovery trends over time.

Comparison of case spikes across regions.

🎯 Learning Highlights

Designing interactive dashboards in Tableau.

Using filters, parameters, and calculated fields for better insights.

Presenting complex datasets visually for storytelling.

Building a professional-level project for analytics portfolio.
